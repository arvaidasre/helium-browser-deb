# Site

This folder contains the assets published to GitHub Pages.

## Files

- `index.html.template`: Template for the landing page.
- `install.sh`: Debian/Ubuntu/Mint installer script.
- `install-rpm.sh`: Fedora/RHEL/CentOS installer script.
- `public/`: CI-generated output (APT/RPM repo + Pages artifacts).

## Publishing

Pages content is generated by CI workflows. Avoid editing `public/` manually.
