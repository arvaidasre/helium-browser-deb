Quick Install

Run a single command to install. Updates are delivered via your package manager.

Ubuntu, Debian, Linux Mint

Terminal

Fedora / RHEL

Fedora, RHEL, CentOS, Rocky

Terminal

Manual Setup

Configure the repository yourself if you prefer manual control.

APT Repository
Terminal 
echo "deb [arch=amd64,arm64] https://arvaidasre.github.io/helium-browser-deb/apt stable main" | sudo tee /etc/apt/sources.list.d/helium-browser.list
sudo apt update
sudo apt install helium-browser
RPM Repository
Terminal 
sudo tee /etc/yum.repos.d/helium.repo << 'EOF'
[helium]
name=Helium Browser
baseurl=https://arvaidasre.github.io/helium-browser-deb/rpm/$basearch
enabled=1
gpgcheck=0
EOF

sudo dnf install helium-browser
